## Contributions
Contributions via pull requests are welcome. Please join our [discord](https://discord.cubi.games) or make an `issue` to discuss any contributions beforehand to make sure they will not conflict with or hinder existing plans.

A good place to start with contributions is to look for `issue`s flagged with the label `help wanted`.

You must hold the rights to any code you contribute, and must agree to licence it under the same licence as the rest of the project in perpetuity, explicitly including any and all changes to the project's licence by project maintainers in the future. The project's licence can always be found at [LICENSE](LICENSE).

### Attribution
Code files may have a copyright notice at the head of the file as follows.
```java
/*
 * SPDX-License-Identifier: AGPL-3.0-only
 * Copyright © 2026 Cubicake.
 * This file is part of RaycastedAntiESP.
 * RaycastedAntiESP is free software: you can redistribute it and/or modify it under the terms of the GNU Affero General Public License v3.0 only, which can be accessed at https://www.gnu.org/licenses/agpl-3.0.html.
 * See README.md for warranty disclaimer and further information.
 */
 ```
If you made a significant contribution to a file with such a header, you may add your name to the list of names in the copyright notice. Contributing project maintainers should be listed first. All other names should be listed in descending order of contribution to the file, adjudicated by `Cubicake` if necessary. If the list of contributors reaches six names, do not add any more names to the list. You may instead append the text `and other contributors`.
